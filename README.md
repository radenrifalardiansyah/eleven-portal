# Eleven Portal

Eleven Digital Indonesia — agency landing page, built with Next.js (App Router), React, Tailwind CSS, Framer Motion, and React Three Fiber for the 3D hero scene.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js 14** (App Router, TypeScript) — single-page app experience with smooth anchor navigation
- **Tailwind CSS** — design tokens for the brand palette
- **Framer Motion** — scroll reveals, 3D pointer tilt, magnetic buttons
- **React Three Fiber / drei / three.js** — animated 3D hero scene
- **Embla Carousel** — case study and stories carousels
- **Lenis** — inertia smooth scrolling
